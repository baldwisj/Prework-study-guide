Git

  • All git commands start with "git"
  • git status: checks what branch we are currently in
  • git checkout -b "branch-name": creates new branch and switches to it
  • git add -A: stages all files
  • Note: Work needs to be staged to be committed or closed out or else loose all work
  • commit -m "commit note": commits code to branch
  • Note: Commit notes neet to be in quotes
