HTML

  • The head element contains information about the webpage
  • The body element represents the visable content shown to the user
  • The use of h1 through h6...ext, is used to distinguish various tiers of headers
  • The use of ul creates unordered lists to be separated by li, which are bullet points
  • h1, ul, and li are used in the body of HTML
  • p in HTML code is used to create a separate text line
  • The "link" tag is used in the head of HTML to create a connection of an external file..ie a css file
  • Along with link, rel specifies the relationship between documents, and herf specifies loaction of a linked file

CSS

  • Use commas to indicate dupication of variables across sections and reduce lines of code
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the inside an element

Git

  • All git commands start with "git"
  • git status: checks what branch we are currently in
  • git checkout -b "branch-name": creates new branch and switches to it
  • git add -A: stages all files
  • Note: Work needs to be staged to be committed or closed out or else loose all work
  • commit -m "commit note": commits code to branch
  • Note: Commit notes neet to be in quotes
